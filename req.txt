ClassTopic
- id: string;
- title: string;
- classId: string;


Assignment
- id: string
- class_id: string (required)
- topic: ClassTopic 
- title: string (required)
- instructions: string (required) <- HTML 
- total_points: number (required, default = 100)
- due_date: number 
- grade_criterias: Array <{
	name: string(required),
	points: string
}>

GradingAssignment
- assignment_id
- class_id
- student_id
- mark: number
- work-attachment {
	link: string,
	type: string,
	submit_at: string,
	last_update: string,
	status: EXPIRED | VALID
}

GET  /classses/:id/assignment?page=1&page_size=10&sort_by=asc&sort_type=topic: get all documents in that class (For STUDENT/TEACHER/ADMIN)
PUT /classses/:id/assignment {data: Assignment} For TEACHER/ ADMIN only
DELETE /classses/:id/assignment
